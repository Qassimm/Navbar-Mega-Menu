# 🎯 Mega Menu – University Website

A responsive and dynamic mega menu built for a Korean university website, focusing on clean UI and smooth user interaction.

## 🚀 Features
- Dynamic menu sections based on user selection  
- Clean and modern UI design  
- Fully responsive layout  

## 🛠️ Tech Stack
- React.js  
- Tailwind CSS  

## 💡 Overview
Developed a mega menu with structured layout and interactive behavior. Different sections are displayed dynamically based on selected navigation options, ensuring a smooth and intuitive user experience.
